# [!UICONTROL 类别数据]数据类型

[!UICONTROL 类别数据]是标准体验数据模型(XDM)数据类型，用于描述与产品类别相关的信息。

![类别数据类型的图表。](../images/data-types/category-data.png)

| 显示名称 | 属性 | 数据类型 | 描述 |
|-----------------|--------------------|-----------|------------------------------------------|
| [!UICONTROL 类别标识符] | `categoryID` | 字符串 | 产品类别的标识符。 |
| [!UICONTROL 类别名称] | `categoryName` | 字符串 | 产品类别的名称。 |
| [!UICONTROL 类别路径] | `categoryPath` | 字符串 | 产品类别的路径。 |

{style="table-layout:auto"}

有关数据类型的更多详细信息，请参阅公共XDM存储库：

* [填充示例](https://github.com/adobe/xdm/blob/master/components/datatypes/categorydata.example.1.json)
* [完整架构](https://github.com/adobe/xdm/blob/master/components/datatypes/categorydata.schema.json)
