# 临时架构

在特定情况下，可能需要创建 [!DNL Experience Data Model] (XDM)架构，其中的字段名称仅由单个数据集使用。 这称为临时架构。 临时架构用于多个数据摄取工作流，包括摄取CSV文件并创建特定类型的源连接器。 有关临时架构的更多信息，包括有关如何在 [!DNL Schema Registry] API，请参阅 [临时模式教程](../tutorials/ad-hoc.md).
