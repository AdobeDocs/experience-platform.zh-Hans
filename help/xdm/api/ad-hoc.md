# 临时架构

在特定情况下，可能有必要创建一个 [!DNL Experience Data Model] (XDM)架构中的字段已命名为仅供单个数据集使用。 这称为临时架构。 临时架构用在多个数据摄取工作流中，包括摄取CSV文件和创建特定类型的源连接器。 有关临时架构的更多信息，包括如何在中创建临时架构的步骤 [!DNL Schema Registry] API，请参阅 [临时架构教程](../tutorials/ad-hoc.md).
