# 临时模式

在特定情况下，可能需要创建一个( [!DNL Experience Data Model] XDM)模式，其中的字段以名称命名，仅供单个数据集使用。 这称为“临时”模式。 点对点模式用于多个数据获取工作流，包括摄取CSV文件和创建某些类型的源连接器。 有关点对点模式的详细信息（包括如何创建点对点的步骤），请参 [阅点对点模式教程](../tutorials/ad-hoc.md)。
