# 临时模式

在特定情况下，可能需要创建一个[!DNL Experience Data Model](XDM)模式，其中的字段以仅由单个数据集使用命名。 这称为临时模式。 临时模式用于多个数据获取工作流，包括获取CSV文件和创建某些类型的源连接器。 有关专门模式的详细信息，包括有关如何在[!DNL Schema Registry] API中创建专门模式的步骤，请参阅[专门应用程序教程](../tutorials/ad-hoc.md)。
