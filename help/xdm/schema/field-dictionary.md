# XDM字段字典


[!DNL Experience Data Model] (XDM)定义了在构成Adobe Experience Platform架构时可以使用的一组标准字段。 使用这些字段会导致跨[!DNL Experience Platform]服务的一致行为，因为所有使用XDM的服务都将明确了解这些字段提供的特定定义。

有关架构构建块的更多信息，包括定义[!DNL Experience Platform]的架构时的原则和最佳实践，请参阅架构组合[的基础知识](composition.md)。

下表包括字段名称（并非所有XDM字段都使用“xdm”进行命名）及其标题、描述、类型和格式（如果需要），以及其对应的XDM类型和可在其中找到字段的资源（架构、架构字段组和数据类型）。

您可以使用[!DNL Schema Editor]中的搜索功能查看这些资源的详细信息，如[架构编辑器教程](../tutorials/create-schema-ui.md#browse)中所示。

>[!NOTE]
>
>如果某个字段被标记为“**DEPRECATED**”，则表示它不应再用于新实施。 该字段仍然可用于实现向后兼容性，但在未来版本中将不再受支持。
