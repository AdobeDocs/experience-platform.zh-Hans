# XDM字段字典


[!DNL Experience Data Model] (XDM)定义一组可在合成Adobe Experience Platform架构时使用的标准字段。 使用这些字段可使 [!DNL Platform] 服务，因为使用XDM的所有服务都将明确了解这些字段提供的特定定义。

请参阅 [架构组合基础知识](composition.md) 有关架构构建基块的更多信息，包括定义架构的原则和最佳实践 [!DNL Experience Platform].

下表包括字段名称（并非所有XDM字段都使用“xdm”命名）、其标题和描述、类型和格式（如果需要），以及其相应的XDM类型和可在其中找到该字段的资源（架构、架构字段组和数据类型）。

您可以在 [!DNL Schema Editor] 查看这些资源的详细信息，如 [模式编辑器教程](../tutorials/create-schema-ui.md#browse).

>[!NOTE]
>
>如果字段被标记为“**已弃用**“，这意味着不应再在新实施中使用它。 字段仍可用于向后兼容，但将来版本将不再支持该字段。
