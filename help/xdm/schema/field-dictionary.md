# 体验数据模型(XDM)字段词典


Experience Data Model(XDM)定义了一组标准字段，这些字段可在构建Adobe Experience Platform模式时使用。 使用这些字段可使平台服务的行为保持一致，因为使用XDM的所有服务都将明确了解这些字段提供的特定定义。

有关模式 [构成的更多信息](composition.md) ，请参阅模式构成的基础知识，包括为Experience Platform定义模式的原则和最佳做法。

下表包括字段名称（并非所有XDM字段都使用“xdm”命名）及其标题和描述、类型和格式（如果需要），以及相应的XDM类型和可在其中找到字段的资源(混合、数据类型和模式)。

您可以使用模式编辑器中的搜索功能视图这些资源的详细信息，如模式编辑器教程中所示。

>[!NOTE] 如果字段标记为“**DEPRECATED**”，则表示该字段不应再用于新实施。 该字段仍可用于向后兼容性，但将来版本中将不再支持。
