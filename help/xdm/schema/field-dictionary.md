# XDM字段字典


[!DNL Experience Data Model] (XDM)定义了在构成Adobe Experience Platform架构时可使用的一组标准字段。 使用这些字段会导致以下各项中的行为一致 [!DNL Platform] 服务，因为使用XDM的所有服务都将明确了解这些字段提供的特定定义。

请参阅 [模式组合基础](composition.md) 有关架构构建块的更多信息，包括定义架构的原则和最佳实践 [!DNL Experience Platform].

下表包括字段名称（并非所有XDM字段都使用“xdm”进行命名空间），以及字段的标题和描述、类型和格式（如果需要），以及其对应的XDM类型和可在其中找到字段的资源（架构、架构字段组和数据类型）。

您可以使用中的搜索功能 [!DNL Schema Editor] 要查看这些资源的详细信息，请参见 [架构编辑器教程](../tutorials/create-schema-ui.md#browse).

>[!NOTE]
>
