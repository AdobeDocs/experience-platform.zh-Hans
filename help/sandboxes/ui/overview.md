# 沙盒 UI 概述

Adobe Experience Platform提供了将单个Experience Platform实例划分为多个单独的独立虚拟环境的虚拟沙箱。 可在[Experience Platform UI](https://platform.adobe.com)中找到沙盒用户界面。

登录到Experience Platform UI后，请继续阅读[沙盒用户指南](user-guide.md)以了解如何在沙盒工作区中执行各种任务的步骤。

有关使用沙盒API执行不同操作的步骤，请参阅[沙盒开发人员指南](../api/getting-started.md)。 有关Experience Platform中沙盒的高级概述，请参阅[概述文档](../home.md)。
