# Experience Platform文档语言支持

提供了多种语言版本的Adobe Experience Platform文档。

要更改显示文档的语言，请在顶部导航中选择语言图标。

![](../images/overview/documentation-language.png)

打开语言下拉列表后，选择要查看文档的语言。

![](../images/overview/documentation-language-select.png)
