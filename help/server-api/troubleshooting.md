# 故障排除

Adobe Experience PlatformEdge Network服务器API允许您从服务中捕获调试信息，因为您的事件是通过Edge Network数据收集管道处理的。

[Experience Platform调试器](https://experienceleague.adobe.com/docs/debugger-learn/tutorials/experience-platform-debugger/introduction-to-the-experience-platform-debugger.html)使用的机制允许您调试基于API的实现。

使用[Assurance](../assurance/home.md)，您可以创建一个调试会话ID，以便进一步在Edge Network请求中用于跟踪事件。
