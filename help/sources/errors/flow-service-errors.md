# 流服务错误消息

流量服务用于收集和集中平台内不同来源的客户数据。 该服务提供了用户界面和RESTful API，可让您轻松设置与各种数据提供商的源连接。 通过这些源连接，您可以验证第三方系统、设置摄取运行的时间，以及管理数据摄取吞吐量。

本文档提供了有关流量服务的错误消息目录、说明和建议的解决方案。
