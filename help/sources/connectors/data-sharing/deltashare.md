# [!DNL Deltashare]

您可以使用[!DNL Deltashare]源连接器建立与[!DNL Databricks Delta Sharing]服务器的安全连接并在Adobe Experience Platform中创建虚拟数据集。 利用此功能，您可以查询虚拟数据集、检查外部表并将其映射到Experience Data Model (XDM)架构，并在Experience Platform应用程序和服务中利用它们。 通过[!DNL Deltashare]，您可以避免数据集的物理摄取，从而减少重复、存储成本和运营开销。
