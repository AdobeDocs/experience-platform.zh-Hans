# [!DNL Kobie]

[!DNL Kobie]提供了一个可用于管理忠诚度计划的整个生命周期的可组合SaaS平台。 [!DNL Kobie]平台通常由营销团队、数据科学家和客户体验经理使用，它包括一个无代码界面，您可以使用它来编排复杂的忠诚度规则。

您可以使用Adobe Experience Platform源目录中的[!DNL Kobie]源从[!DNL Kobie]帐户中摄取批次和流式会员数据。

- [[!DNL Kobie]流](../../tutorials/ui/create/loyalty/kobie-streaming.md)
