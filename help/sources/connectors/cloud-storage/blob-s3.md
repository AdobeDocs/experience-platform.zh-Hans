# Azure Blob和Amazon S3连接器

Adobe Experience Platform为AWS、[!DNL Google Cloud Platform]和[!DNL Azure]等云提供商提供本机连接。 您可以将这些系统中的数据导入[!DNL Platform]。

云存储源可以将您自己的数据导入[!DNL Platform]，而无需下载、格式化或上传。 收录的数据可以格式化为XDM JSON、XDM Perface或分隔。 流程的每个步骤都集成到源工作流中。 [!DNL Platform] 允许您从和S3中 [!DNL Azure Blob] 通过批量导入数据。

有关[!DNL Azure Blob]连接器的详细信息，请阅读[[!DNL Azure Blob] 连接器概述](./blob.md)。

有关[!DNL Amazon S3]连接器的详细信息，请阅读[[!DNL Amazon S3] 连接器概述](./s3.md)。
