# Azure Blob和Amazon S3连接器

Adobe Experience Platform为AWS等云提供商提供本机连接， [!DNL Google Cloud Platform]、和 [!DNL Azure]. 您可以将来自这些系统的数据导入 [!DNL Platform].

云存储源可以将您自己的数据引入 [!DNL Platform] 无需下载、格式化或上传。 引入的数据可以格式化为XDM JSON、XDM Parquet或分隔。 该过程的每个步骤都集成到源工作流中。 [!DNL Platform] 允许您从以下位置引入数据 [!DNL Azure Blob] 和S3。

欲知关于 [!DNL Azure Blob] 连接器，请阅读 [[!DNL Azure Blob] 连接器概述](./blob.md).

欲知关于 [!DNL Amazon S3] 连接器，请阅读 [[!DNL Amazon S3] 连接器概述](./s3.md).
