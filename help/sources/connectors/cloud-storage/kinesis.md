# （测试版）Amazon Kinesis连接器

>[!NOTE]
>Amazon Kinesis连接器为测试版。 有关使用 [测试版标记](../../home.md#terms-and-conditions) 的连接器的更多信息，请参阅源概述。

Adobe Experience Platform为AWS、Google Cloud Platform和Azure等云提供商提供本机连接。 您可以将这些系统中的数据引入Platform。

云存储源可以将您自己的数据引入Platform中，而无需下载、格式化或上传。 摄取的数据可格式化为XDM JSON、XDM镶木地板或分隔。 流程的每个步骤都集成到源工作流中。 Platform允许您实时从Amazon Kinesis导入数据。
