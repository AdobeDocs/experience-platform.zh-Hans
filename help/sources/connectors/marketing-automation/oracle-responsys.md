# (Beta) [!DNL Oracle Responsys] 源

>[!NOTE]
>
>此 [!DNL Oracle Responsys] 源为测试版。 请参阅 [源概述](../../home.md#terms-and-conditions) 有关使用测试版标记源的更多信息。

Adobe Experience Platform允许从外部源摄取数据，同时让您能够使用来构建、标记和增强传入数据 [!DNL Platform] 服务。 您可以从各种来源(如Adobe应用程序、基于云的存储、数据库和许多其他来源)中摄取数据。

[!DNL Oracle Responsys] 是一个跨渠道营销活动管理平台，可让您大规模提供高级智能，并构建符合客户特定兴趣和偏好设置的自定义消息。

Experience Platform支持从第三方营销自动化系统中引入数据。 对营销自动化提供商的支持包括 [!DNL Oracle Responsys].

## IP地址允许列表

在使用源连接器之前，必须将IP地址列表添加到允许列表中。 未能将特定于地区的IP地址添加到允许列表中，可能会导致使用源时出现错误或性能不佳。 请参阅 [IP地址允许列表](../../ip-address-allow-list.md) 页面，以了解更多信息。
