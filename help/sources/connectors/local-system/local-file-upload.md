# 本地文件上传连接器

Adobe Experience Platform允许从外部源和本地系统中摄取数据，同时让您能够使用以下工具来构建、标记和增强传入数据： [!DNL Platform] 服务。 您可以从各种来源(如Adobe应用程序、基于云的存储、数据库和许多其他来源)中摄取数据。

Experience Platform支持从本地系统中摄取数据，从而允许您通过使用源连接器将本地文件传送到Experience Platform。
