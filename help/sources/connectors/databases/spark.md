# (Beta) [!DNL Apache Spark] 日期 [!DNL Azure HDInsights] 连接器

>[!NOTE]
>
>此 [!DNL Apache Spark] 日期 [!DNL Azure HDInsights] 连接器处于测试阶段。 请参阅 [源概述](../../home.md#terms-and-conditions) 有关使用Beta标记的连接器的更多信息。

Adobe Experience Platform允许从外部源摄取数据，同时让您能够使用来构建、标记和增强传入数据 [!DNL Platform] 服务。 您可以从各种来源(如Adobe应用程序、基于云的存储、数据库和许多其他来源)中摄取数据。

[!DNL Experience Platform] 支持从第三方数据库引入数据。 [!DNL Platform] 可以连接到不同类型的数据库，如关系数据库、NoSQL数据库或数据仓库数据库。 对数据库提供程序的支持包括 [!DNL Apache Spark] 日期 [!DNL Azure HDInsights].

## IP地址允许列表

在使用源连接器之前，必须将IP地址列表添加到允许列表中。 未能将特定于区域的IP地址添加到允许列表中，可能会导致使用源时出现错误或性能不佳。 请参阅 [IP地址允许列表](../../ip-address-allow-list.md) 页面以了解更多信息。
