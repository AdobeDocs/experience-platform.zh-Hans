# [!DNL HTTP API] 连接器

Adobe Experience Platform允许从外部源摄取数据，同时让您能够使用构建、标记和增强传入数据 [!DNL Platform] 服务。 您可以从各种源摄取数据，如Adobe应用程序、基于云的存储、数据库和许多其他源。

您可以使用 [!DNL HTTP API] 源连接器将您的数据流传输到平台。 的 [!DNL HTTP API] 源受支持 [!DNL Data Prep] 函数，允许您将任何不符合XDM的数据映射到符合XDM的数据集。
