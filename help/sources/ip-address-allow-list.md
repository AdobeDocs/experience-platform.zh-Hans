# IP地址允许列表

您可以通过网络防火墙定义网络访问控制。 通过指定适当的源IP范围，您可以允许数据传输服务的通信量。 在使用源连接器之前，必须将以下IP地址添加到允许列表。 如果无法向允许列表添加特定于区域的IP地址，则在使用源时可能会导致错误或性能不佳。

## 美国东部地区

- `20.42.2.0/23`
- `20.42.4.0/26`
- `20.42.64.0/28`
- `20.49.111.0/29`
- `40.71.14.32/28`
- `40.78.229.96/28`
