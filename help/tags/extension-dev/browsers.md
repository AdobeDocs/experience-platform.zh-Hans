# 支持的浏览器

Adobe Experience Platform中的标记运行时（交付到客户端网站）与多个浏览器兼容：

- Chrome（最新版本）
- Safari（最新版本）
- Firefox（最新版本）
- Edge（最新版本）
- iOS Safari（最新版本）
- Android Chrome（最新版本）

运行时中包含的所有扩展代码也应该尽量支持这些浏览器。

数据收集UI可支持以下浏览器：

- Chrome（最新版本）
- Safari（最新版本）
- Firefox（最新版本）
- Edge（最新版本）

数据收集UI中显示的任何扩展代码也应该尽量支持这些浏览器。
