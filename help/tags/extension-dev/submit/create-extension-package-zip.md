# 创建扩展包zip文件。

>[!NOTE]
>
>经过品牌重塑，Adobe Experience Platform Launch 已变为 Adobe Experience Platform 中的一套数据收集技术。因此，产品文档中的术语有一些改动。有关术语更改的综合参考，请参阅以下[文档](../../term-updates.md)。

要准备扩展包以在Adobe Experience Platform中上传，请参阅[@adobe/reactor-packager](https://www.npmjs.com/package/@adobe/reactor-packager)命令行工具的API文档。 它将以预期格式输出准备好的zip文件，用于[上载扩展](./upload-and-test.md)。
