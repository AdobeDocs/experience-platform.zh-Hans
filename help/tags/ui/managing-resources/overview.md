# 管理资源

在Experience Platform中的标记上下文中，“资源”一词是指以下任意项目：

* [扩展](extensions/overview.md) - 扩展可提供数据元素类型和规则以供您使用。
* [数据元素](data-elements.md) - 使用数据元素可以轻松引用位于其他位置的任何数据。它们会创建可在规则和扩展配置中使用的快捷方式。当基础数据的位置发生更改时，可以更新它们以指向新位置，而无需更新单个规则或扩展。
* [规则](rules.md) - 规则是规则组件（事件、条件和操作）的集合，用于控制已部署标记的行为。

您可以通过选择上面的链接，详细了解每种资源类型。
