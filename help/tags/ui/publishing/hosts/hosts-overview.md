# 主机概述

扩展、规则和数据元素是构建块。如果您希望应用程序执行一些操作，则需要将这些构建块添加到库中，然后将库“构建”到内部版本中。这些内部版本将被交付到托管位置。

主机定义可以交付内部版本的可用目标。主机分为两种类型：

* [Managed by Adobe](./managed-by-adobe-host.md)
* [SFTP](./sftp-host.md)

有关每种主机类型的更多信息，请查看上述链接。

同一主机可用于一个资产中的多个环境。

有关帮助您在自托管与由Adobe为您管理托管之间做出决定的更多信息，请阅读[自托管指南](./self-hosting-libraries.md)。
