# Adobe Media Analytics (3.x SDK) for Audio and Video发行说明

## 2021年4月5日

### Adobe Media Analytics (3.x SDK) for Audio and Video扩展3.0.2

* 修复了媒体收集API请求中的“Content-Type”HTTP标头。

## 2020 年 5 月 10 日

### Adobe Media Analytics (3.x SDK) for Audio and Video扩展3.0.0

* Media SDK v3.0.0 包装器
* 使用媒体收集 API 跟踪媒体
* 支持播放器状态跟踪

[Adobe Media Analytics (3.x SDK) for Audio and Video 扩展](./overview.md)添加了核心 Media Analytics 3.x JavaScript 库。此库提供了将媒体跟踪到启用了标记的网站或项目的功能。
