# Adobe Media Analytics (3.x SDK) for Audio and Video 发行说明

>[!NOTE]
>
>Adobe Experience Platform Launch已經過品牌重塑，現在是Adobe Experience Platform中的一套資料收集技術。 因此，产品文档中的术语有一些改动。有关术语更改的综合参考，请参阅以下[文档](../../../term-updates.md)。

## 2021年4月5日

### Adobe Media Analytics (3.x SDK) for Audio and Video 扩展 3.0.2

* 修正媒體收集API請求中的「Content-Type」HTTP標頭。

## 2020 年 5 月 10 日

### Adobe Media Analytics (3.x SDK) for Audio and Video 扩展 3.0.0

* Media SDK v3.0.0 包装器
* 使用媒体收集 API 跟踪媒体
* 支持播放器状态跟踪

[Adobe Media Analytics (3.x SDK) for Audio and Video 扩展](./overview.md)添加了核心 Media Analytics 3.x JavaScript 库。此程式庫提供追蹤已啟用標籤之網站或專案的媒體功能。
