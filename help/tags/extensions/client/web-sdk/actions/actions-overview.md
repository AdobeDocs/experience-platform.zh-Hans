# 操作类型

本页列出了标记规则中支持的所有操作。 要创建或编辑标记规则中的操作，请执行以下操作：

1. 使用您的Adobe ID凭据登录[experience.adobe.com](https://experience.adobe.com)。
1. 导航到&#x200B;**[!UICONTROL Data Collection]** > **[!UICONTROL Tags]**。
1. 选择所需的标记属性。
1. 导航到&#x200B;**[!UICONTROL Rules]**，然后选择所需的规则。
1. 在[!UICONTROL Actions]下，选择现有操作或创建操作。
1. 将[!UICONTROL Extension]下拉字段设置为&#x200B;**[!UICONTROL Adobe Experience Platform Web SDK]**，然后选择所需的[!UICONTROL Action Type]。

Web SDK标记扩展提供了以下操作：

* [**[!UICONTROL Apply propositions]**](apply-propositions.md)：在单页应用程序中渲染建议而不增加量度。
* [**[!UICONTROL Apply response]**](apply-response.md)：根据Edge Network的响应执行操作。
* [**[!UICONTROL Evaluate rulesets]**](evaluate-rulesets.md)：手动触发规则集评估。
* [**[!UICONTROL Get Media Analytics tracker]**](get-media-analytics-tracker.md)：将旧版Media API导出到窗口对象。
* [**[!UICONTROL Redirect with identity]**](redirect-with-identity.md)：允许在您拥有的域之间共享访客标识符。
* [**[!UICONTROL Send event]**](send-event.md)：将事件数据发送到Edge Network。
* [**[!UICONTROL Send media event]**](send-media-event.md)：将媒体数据发送到Edge Network。
* [**[!UICONTROL Set consent]**](set-consent.md)：为访客设置所需的同意。
* [**[!UICONTROL Update variable]**](update-variable.md)：修改[变量](../data-element-types.md#variable)数据元素。
