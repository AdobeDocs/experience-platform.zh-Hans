# Adobe Experience Platform Web SDK标记扩展概述

Adobe Experience Platform Web SDK标记扩展通过Adobe Experience Platform Edge Network从Web资产向Adobe Experience Cloud发送数据。 Experience Platform Web SDK扩展允许将数据流式传输到平台、同步身份、选择加入和自动收集上下文数据。

请参阅[快速入门](getting-started.md)，了解如何设置和部署此标记扩展以开始收集数据。

观看以下视频以了解Adobe Experience Platform [!DNL Web SDK]和[!DNL Edge Network]的实际操作情况，只需一次调用即可将数据发送到[!DNL Experience Platform]、[!DNL Analytics]、[!DNL Audience Manager]和[!DNL Target]。
