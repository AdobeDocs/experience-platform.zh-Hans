# AEM资产分析发行说明

## 2020 年 9 月 9 日

**AEM 资产洞察扩展 1.0.3**

**功能**

- 这项扩展旨在与 AEM 资产洞察结合使用。
- 更具体地说，它可以替换 ***pageTracker*** 进程和嵌入代码。
- 配置并激活这项扩展后，会将AEM资产的“展示次数”和“单击次数”量度发送到Adobe Analytics，随后，这些量度将自动导入回AEM的资产分析报表中。

**错误修复**

- 无
