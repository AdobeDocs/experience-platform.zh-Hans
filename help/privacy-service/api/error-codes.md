# Privacy Service错误代码 {#privacy-service-error-codes}

使用此参考可识别Privacy Service作业结果、诊断故障以及确定在&#x200B;**Adobe Experience Platform**&#x200B;中提交或监视隐私作业时相应的后续步骤。 要了解如何创建、提交和监视隐私作业，请参阅[隐私作业端点指南](./privacy-jobs.md)或[Privacy Service UI用户指南](../ui/user-guide.md)。

Privacy Service错误代码是一份稳定的公共合同。 每个错误代码唯一标识故障或完成状态，您可以依赖该状态进行程序化处理和操作工作流。

构建自动化或监控工作流时需遵守以下保证：

* 错误代码在发布后是稳定的。
* 为了提高清晰度，错误消息可能会发生变化，但代码值不会发生变化。
* 随着时间的推移，可能会添加新的错误代码；现有代码不会重新利用。

使用错误代码（而非消息文本）实现自动化或决策逻辑。 有关在不依赖轮询或消息字符串的情况下高效处理隐私作业、监视作业状态和处理错误的指导，请参阅[Privacy Service最佳实践](../best-practices.md)。

## 错误响应格式 {#error-response-format}

Privacy Service在作业和请求响应中返回错误信息。 响应在响应有效载荷中包含数值错误代码和人类可读的消息。

错误代码将传达权威结果。 该消息提供了用于疑难解答的其他上下文。

本文档描述了每个错误代码的含义和用途。 有关字段级响应架构和请求详细信息，请参阅[Privacy Service API文档](https://developer.adobe.com/experience-platform-apis/references/privacy-service/)。

## 错误域 {#error-domains}

错误代码按功能域分组，以帮助您更快地诊断问题。

本文档中使用的域包括：

* **请求验证**：请求格式错误或包含无效值。 有关请求结构和验证要求，请参阅[隐私作业终结点指南](./privacy-jobs.md)。
* **授权和设置**：您的组织或用户缺少所需的访问权限。 请参阅[管理权限](../permissions.md)以查看基于角色的权限要求。
* **标识和适用性**：标识符或命名空间不适用于该请求。 有关支持的身份类型和命名空间要求，请参阅隐私请求的[身份数据](../identity-data.md)。
* **速率限制**：提交卷超过平台限制。 发生此错误时，请降低提交率并重试。
* **数据访问和处理**：系统无法访问或处理请求的数据。 有关常见原因和修正步骤，请参阅[疑难解答指南](../troubleshooting-guide.md)。
* **加密和密钥管理**：所需的加密密钥不可用。 有关密钥访问、配置和恢复指导，请参阅[客户管理的密钥](../../landing/governance-privacy-security/customer-managed-keys/overview.md)。
* **作业执行状态**：作业已完全完成、部分完成或失败。 有关作业状态类别及其含义的说明，请参阅[隐私作业端点指南](./privacy-jobs.md#status-categories)。

>[!NOTE]
>
>域分配反映的是错误代码的意图，而不是内部服务边界。

## 错误代码引用 {#error-code-reference}

下表列出了所有公共Privacy Service错误代码。

| 错误代码 | HTTP状态 | 标题 | 描述 |
| ---------- | ----------- | ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------| 
| 1000-400 | 400 | 格式设置错误 | 指定应用程序的一个或多个数据值存在格式问题。 检查作业详细信息以了解其他信息。 |
| 1001-400 | 400 | 未授权 | 您的组织未配置。 请联系您的管理员以获取更多信息。 |
| 1010-400 | 400 | 缺少权限 | 您没有执行此操作所需的权限。 联系管理员以请求获取访问权限。 |
| 1020-400 | 400 | 上传和存档失败 | 上载和归档访问数据时出现问题。 请上传访问数据并重试。 |
| 1021-400 | 400 | 作业失败 | 根据请求创建的一个或多个隐私作业失败。 有关其他信息，请查看失败作业的详细信息。 |
| 1022-400 | 400 | 数据访问失败 | 访问指定数据时出现问题。 有关其他信息，请查看作业详细信息。 |
| 1023-400 | 400 | 数据访问失败 | 访问或定位指定的数据集ID时出现问题。 请验证提供的ID是否有效，然后重试。 |
| 1024-400 | 400 | 意外错误 | 发生意外错误。 有关其他信息，请查看作业详细信息。 |
| 1030-400 | 400 | 超出文档速率限制 | 工作负载超出了文档速率限制。 请降低提交率，然后重试。 |
| 1040-400 | 400 | 密钥加密访问失败 | 无法处理数据，因为数据存储已加密，并且密钥访问权限已撤销。 请联系您的密钥库管理员以恢复对客户密钥的访问权限。 |
| 6000-200 | 200 | 成功 | 作业已成功完成。 查看作业详细信息以确认已处理的记录和结果。 |
| 6051-200 | 200 | 未配置 | 您的组织未针对所请求的应用程序进行配置。 该请求不适用。 |
| 6052-200 | 200 | 未找到用户ID | 未找到某些用户ID，未知用户ID不适用于此产品。 请确保用户ID有效，然后重试。 |
| 6053-200 | 200 | 命名空间无效 | 提供的身份命名空间对此应用程序无效。 使用系统识别的命名空间。 |
| 6054-200 | 200 | 部分完成 | 作业已完成，无法获取适用的数据，但某些数据不适用于该请求。 有关其他信息，请查看作业详细信息。 |
