# [!UICONTROL 配置文件] 仪表板 {#profile-dashboard}

Adobe Experience Platform用户界面(UI)提供了一个功能板，通过该功能板，您可以查看有关 [!DNL Real-Time Customer Profile] 数据，在每日快照期间捕获。

有关如何访问和与 [!UICONTROL 配置文件] 要了解更多有关仪表板中显示的可用量度的信息，请访问 [[!UICONTROL 配置文件] 仪表板指南](../../dashboards/guides/profiles.md).

有关Experience Platform用户界面中所有功能板功能的概述，请先阅读 [功能板概述](../../dashboards/home.md).
