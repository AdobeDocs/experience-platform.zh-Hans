# [!UICONTROL 个人资料]仪表板 {#profile-dashboard}

Adobe Experience Platform用户界面(UI)提供了一个仪表板，通过该仪表板可以查看有关每日快照期间捕获的[!DNL Real-Time Customer Profile]数据的重要信息。

有关如何在UI中访问和与[!UICONTROL 配置文件]仪表板进行交互的详细说明，以及要详细了解仪表板中显示的可用量度，请访问[[!UICONTROL 配置文件]仪表板指南](../../dashboards/guides/profiles.md)。

有关Experience Platform用户界面中所有仪表板功能的概述，请先阅读[仪表板概述](../../dashboards/home.md)。
