Data Distiller是一个包，它包含Adobe Experience Platform的部分功能。 此资源包旨在通过查询服务中执行的批量查询，为实时客户档案或分析数据用例准备摄取后数据。 请参阅 [数据Distiller概述](../data-distiller/overview.md) 以了解有关Data Distiller包的更多详细信息。
