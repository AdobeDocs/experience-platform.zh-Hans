# 将客户端连接到查询服务

本节介绍如何从各种桌面客户端应用程序连接到查询服务，以及如何验证这些连接。 查询服务使用Postgres协议，因此本节中的说明将说明如何使用Postgres工具和驱动程序连接和编写查询。

为下列客户端提供了说明：

- [Aqua Data Studio](./aqua-data-studio.md)
- [Looker](./looker.md)
- [Postico(Mac)](./postico.md)
- [Power BI(PC)](./power-bi.md)
- [PSQL](./psql.md)
- [RStudio](./rstudio.md)
- [塔布洛](./tableau.md)
