# 查询服务SQL引用

[!DNL Query Service] 提供使用标准ANSI SQL的 `SELECT` 语句和其他有限命令。 [!DNL Query Service] 不包括数据定义语言或数据操作语言功能。 通过服务查询的数据来自 [!DNL Experience Data Lake]，其中包含基本不可变的数据集。
