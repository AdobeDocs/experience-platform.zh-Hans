# SQL引用

[!DNL Query Service] 提供对语句和其他有限命令使 `SELECT` 用标准ANSI SQL的能力。 [!DNL Query Service] 不包括数据定义语言或数据操作语言功能。 通过服务查询的数据来自 [!DNL Experience Data Lake]基本不可改变的数据集。
