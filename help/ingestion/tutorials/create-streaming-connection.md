# 使用API创建流连接

要将流数据开始到Adobe Experience Platform，需要流连接注册。 注册流连接时，您需要提供一些关键详细信息，如流数据源。

注册流连接后，作为数据生成者，您将拥有一个唯一的URL，可用于将数据流化到平台。

要了解如何创建经过身份验证的流连接，请阅读[HTTP API源API指南](../../sources/tutorials/api/create/streaming/http.md)。
