# 使用UI创建流连接

要将流数据开始到[!DNL Experience Platform]，必须首先创建流HTTP连接。 创建流连接时，您需要提供关键详细信息，如流数据源，以及您是否打算从受信任的（已验证）源或不受信任的（未验证）源发送数据。

注册流连接后，您将有一个唯一的URL，它可用于将数据流化到[!DNL Platform]。

要了解如何创建流连接，请阅读[HTTP API源UI指南](../../sources/tutorials/ui/create/streaming/http.md)。
