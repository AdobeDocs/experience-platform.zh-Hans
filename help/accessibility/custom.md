# 自定义辅助功能解决方案，用于Experience Platform

Adobe Experience Platform不断得到增强，以满足所有类型用户的需求，并遵守包括患有视觉、听觉、移动或其他障碍的个人在内的全球标准。 本文档概述了Experience Platform用户界面中的自定义辅助功能解决方案。

## 主页和用户界面概述

Experience Platform用户界面符合常规文本、图形和UI组件所需的对比度。 此外，还选择了用户界面的颜色以支持所有用户（包括患有视觉障碍的用户）的辅助功能。

在Platform中，可单击或可与指针一起操作的UI元素也可以使用键盘参与。 这包括左侧导航、视频播放器、表格等。

Experience Platform努力满足国际无障碍标准，包括Web内容无障碍准则2.1 A级和AA级，以及Web无障碍倡议 — 无障碍富因特网应用程序(WAI-ARIA)Web标准。

![Adobe Experience Platform UI的主页。](images/homepage.png)

## 左侧导航栏

Experience PlatformUI中的左侧导航是可通过键盘访问的，并在正常、悬停和选择状态下提供符合辅助功能标准的颜色对比度。

从主屏幕中，用户可以使用Tab键进入左侧导航。 选择 **Shift +选项卡** 将用户返回到主屏幕。

![Experience Platform左侧导航。](images/left-navigation-select.png)

左侧导航聚焦后， **选项卡** 会使用户进行展开和折叠交互。 展开或折叠左侧导航的功能将通过 **Enter（返回）**.

![Experience Platform左侧导航折叠。](images/left-navigation-collapse.png)

焦点位于左侧导航中时，向上和向下箭头键可导航到导航中的每个项目并持续循环（换言之，在用户选项卡远离左侧导航之前，焦点不会移开）。 选中时，将显示导航项目的焦点。 显示当前选定内容时，会突出显示和加粗文本。 选择左侧导航项时， **Enter（返回）** 在右侧面板中打开选定的UI项目，但是焦点仍保留在左侧导航中，直到用户选项卡离开为止。

![Experience Platform左侧导航，并选择源。](images/left-navigation-sources.png)

平台中的某些功能并未为所有用户启用。 这些项目显示在导航中，但无法选择。 使用键盘导航时，箭头导航时会跳过这些项目，无法使用 **Enter（返回）**.

![无法选择Experience Platform左侧导航中未为用户启用的部分。](images/left-navigation-sections-disabled.png)

## 嵌入式视频对话框

通过使用键盘导航突出显示和选择可用的视频链接，可以在Experience Platform内查看视频。 这会在平台UI中打开一个嵌入式视频对话框。

![选定元素周围显示蓝色边框，指示已应用焦点。](images/profile-overview-tab.png)

## 视频对话框键盘辅助功能

也可以使用键盘导航嵌入式视频对话框。 下表概述了可用于嵌入式视频对话框的完整键盘导航。

| 对话框元素 | 键盘辅助功能 | 描述 |
|---|---|---|
| 播放并暂停 | 选项卡<br/>空格键 | 使用 **选项卡** 来设置“焦点”按钮。 **空格键** 开始视频播放并暂停视频播放。 |
| 洗涤器 | 选项卡<br/>向左箭头<br/>向右箭头 | 播放视频时，请使用 **选项卡** 聚焦洗涤器。 在洗涤器聚焦后， **向左和向右箭头键** 分别先后跳过视频播放和后退5秒。 |
| 静音 | 选项卡<br/>空格键 | 使用 **选项卡** 来集中静音音量元素。 使用 **空格键** 使视频播放静音或取消静音。 |
| 数量 | 选项卡<br/>向左箭头<br/>向右箭头 | 使用 **选项卡** 以关注卷元素。 **向左和向右箭头键** 分别上下移动卷。 |
| [!UICONTROL 隐藏式字幕] (&quot;cc&quot;) | 选项卡<br/>输入<br/>向上箭头<br/>向下箭头 | **选项卡** to [!UICONTROL 隐藏式字幕] (“cc”)元素。 使用 **输入** 打开菜单， **向上和向下箭头键** 为字幕选择语言。 **输入** 确认您的选择。 |
| [!UICONTROL 质量] | 选项卡<br/>输入<br/>向上箭头<br/>向下箭头 | 使用 **选项卡** 聚焦 [!UICONTROL 质量] 元素。 使用 **输入** 打开菜单，然后 **向上和向下箭头键** 来选择视频质量。 **输入** 确认您的选择。 |
| 全屏 | 选项卡<br/>空格键或Enter<br/>Escape | 使用 **选项卡** 以集中显示全屏元素。 使用 **空格键或Enter** 激活全屏视图。 **Escape** (“esc”)退出全屏模式。 |
| 关闭 | 选项卡<br/>空格键或Enter | 使用 **选项卡** 来集中关闭按钮。 使用 **空格键或Enter** 键退出视频对话框。 |

>[!NOTE]
>
>在播放过程中，可随时使用转义(“esc”)键关闭嵌入的视频对话框。

## 文件拖放

在Experience Platform中，所有文件选择拖放区域都可通过键盘访问。 使用 **选项卡** 突出显示 **[!UICONTROL 选择文件]** 和使用 **输入或空格键** 若要选择，将调用操作系统的文件选择UI。

上传文件后，删除图标变为键盘可导航，以删除选定的文件并上传新文件。 用户可以使用 **选项卡** 要集中查看删除图标和 **输入或空格键** 来选择它。 文件被删除后， **[!UICONTROL 选择文件]** 会自动聚焦并能够选择。

或者，如果上传的文件格式不正确，则会显示错误图标以及错误消息，并且 **[!UICONTROL 选择文件]** 按钮处于焦点状态并可进行选择。

![带有错误消息的文件拖放区域和焦点中的“选择文件”按钮。](images/drag-and-drop.png)

使用鼠标选择拖放区域也会调用文件选择UI，或者使用鼠标的用户可以选择文件并拖动到区域以开始上传。

![当鼠标用户将文件拖放到焦点区域时，文件拖放区域会将其拖入区域。](images/drag-and-drop-mouse-over.png)

## 表浏览

Experience Platform用户界面中的所有表都可通过键盘访问。 可通过一系列键盘快捷键浏览表行和表列并与之交互：

* 从表标题中，使用 **向下箭头** 浏览表格。 在通过 **选项卡**，则可以使用 **空格键**.
* **向上和向下箭头键** 在表格的行中上下移动。
* 选择行或聚焦行时，使用 **输入** 行上的右侧边栏提供了详细信息。
* 选择行或聚焦行后，使用 **箭头键** 在行中移动每个项目。
* 使用 **输入** 来选择行中的项目。 如果必须打开新窗口，则具有屏幕阅读器的用户会收到警报。
* 当您缩放到200%或更多时，您可以看到 **边栏检查器** 图标，以便为表格提供更多查看空间。

![当用户缩放到200%时，边栏检查器图标处于焦点。](images/rail-inspector.png)

### 浏览表格键盘辅助功能

| 键盘辅助功能 | 描述 |
|---|---|
| HOME（函数+左箭头） | 聚焦行时，会将用户引导至行中的第一个项目 |
| END（函数+向右箭头） | 聚焦行时，会将用户引向行中的最后一项 |
| 页面设置 | 向上遍历表中的10行（每页） |
| 向下页面 | 向下遍历表中的10行（每页） |
| 控制+主页 | 转到表中的第一行 |
| 控制+结束 | 每页转到表格中的首个工作 |

## 架构编辑器UI

可通过以下功能访问架构编辑器UI:

* 架构编辑器支持键盘导航，包括使用 **选项卡** 导航到UI元素。
* **选项卡** 输入搜索字段，然后进入架构树。
* 架构树支持使用箭头键在架构树UI中导航
   * **向上和向下箭头** 可以用来遍历树。
   * **左右箭头** 可用于展开和折叠节点或在架构树上的内联操作之间移动。
* **Enter（返回）** 在右侧的“详细信息”面板中激活单个节点详细信息。
* 的 **主页** 键返回到树的顶部。
* 的 **结束** 键导航到树的底部。
* 架构树还包含用于屏幕阅读器的ARIA标签。

## 区段生成器UI

使用区段生成器UI创建、编辑区段并与Experience Platform中的区段交互时，以下功能可改进辅助功能：

* 可通过键盘导航访问区段生成器UI。
* 屏幕阅读器应识别标题的标记标记，并可以读出标题及其级别。
* 其他辅助型技术可以使用正确编码的标题来显示大纲或替代视图，来更改页面的可视显示。

## 查询服务编辑器

查询服务编辑器中提供了以下辅助功能：

* 查询服务编辑器UI中的颜色对比度符合辅助功能合规性。
* 编辑器UI外部支持键盘导航。 编辑器UI是嵌入的代码镜像。

## 源和目标中的“系统视图”选项卡

浏览 **[!UICONTROL 系统视图]** 在源和目标中，以下功能改进了辅助功能：

* **选项卡** 将焦点放在第一个源连接卡上
   * **选项卡** 再次关注卡内部的按钮
   * 选择 **输入** 在卡内激活行动动员按钮
* 选择 **输入** 在连接卡上，还会激活右边栏中的更多详细信息
   * 激活右边栏后，焦点将设置到该区域。 **选项卡** 重点 **关闭** 的上界。 选择 **选项卡** 再次通过右边栏面板移动焦点
   * 如果有多个源连接卡， **选项卡** 通过连接
   * 使用 **箭头键（上、下、左和右）** 在源列表中移动
   * 选择 **选项卡** 在右边栏面板上设置焦点
