## 检索受众模板 {#retrieve}

您可以通过向`/authoring/audience-templates`端点发出`GET`请求来检索现有的受众模板。

**API格式**

使用以下API格式检索您帐户的所有受众模板。

```http
GET /authoring/audience-templates
```

使用以下API格式检索由`{INSTANCE_ID}`参数定义的特定受众模板。

```http
GET /authoring/audience-templates/{INSTANCE_ID}
```

以下两个请求检索您的IMS组织或特定受众模板的所有受众模板，具体取决于您是否在该请求中传递`INSTANCE_ID`参数。

选择下面的每个选项卡以查看相应的有效负载。

>[!BEGINTABS]

>[!TAB 检索所有受众模板]

以下请求将基于[!DNL IMS Org ID]和沙盒配置检索您有权访问的受众模板列表。

+++请求

```shell
curl -X GET https://platform.adobe.io/data/core/activation/authoring/audience-templates \
 -H 'Authorization: Bearer {ACCESS_TOKEN}' \
 -H 'x-gw-ims-org-id: {ORG_ID}' \
 -H 'x-api-key: {API_KEY}' \
 -H 'x-sandbox-name: {SANDBOX_NAME}'
```

+++
