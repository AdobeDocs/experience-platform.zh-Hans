# 监视数据流

[!UICONTROL 目标] 允许您从Adobe Experience Platform向无数外部合作伙伴激活数据。

数据流是跨数据移动数据的数据作业的表示形式 [!DNL Platform]. 这些数据流是跨不同的服务配置的，有助于将数据从源连接器移动到目标数据集，然后数据集会被利用 [!DNL Identity Service] 和[!DNL Real-Time Customer Profile] 最终激活到目标之前。

要了解有关监控目标数据流的更多信息，请阅读 [在UI中监控目标的数据流](../../dataflows/ui/monitor-destinations.md) 教程。
