# 监测数据流

[!UICONTROL 目标]允许您将数据从Adobe Experience Platform激活到无数外部合作伙伴。

数据流是跨[!DNL Platform]移动数据的数据作业的表示形式。 这些数据流在不同的服务中配置，有助于将数据从源连接器移动到目标数据集，然后由[!DNL Identity Service]和[!DNL Real-Time Customer Profile]使用它，最后激活到目标。

要了解有关监视目标的数据流的更多信息，请阅读UI中的[监视目标的数据流](../../dataflows/ui/monitor-destinations.md)教程。
