# 目标目录概述

目标目录提供了可用于接收数据的可用目标平台列表。 这些目标平台包括Adobe解决方案、广告平台、调查平台、电子邮件营销平台、Experience Platform扩展等。

浏览目录以确定哪些目标平台最适合您的广告和个性化用例，然后设置流程以将区段激活到目标。
