# [!DNL Data Management Platform] (DMP)目标概述  {#dmp-destinations}

[!DNL Data Management Platforms] (DMP)使广告商、出版商和代理商能够构建独特的受众用户档案，确定其最有价值的细分，并在任何数字渠道中使用它们。这使您能够真正创建消费者用户档案的360视图，构建并激活有意义的受众。
