# [!DNL Data Management Platform] (DMP)目标概述 {#dmp-destinations}

[!DNL Data Management Platforms] (DMP)使广告商、出版商和代理能够构建独特的受众配置文件，识别其最有价值的区段，并在任何数字渠道中使用它们。这样，您就可以创建消费者配置文件的真实360视图，并构建和激活有意义的受众。
