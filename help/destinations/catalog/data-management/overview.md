# [!DNL Data Management Platform] (DMP)目标概述 {#dmp-destinations}

[!DNL Data Management Platforms] (DMP)使广告商、出版商和代理机构能够构建独一无二的受众配置文件，识别他们最有价值的区段，并在任何数字渠道中使用这些区段。 这允许以真实360视图查看使用者配置文件，构建和激活有意义的受众。

目前，[!DNL Adobe Experience Platform]支持下面列出的数据管理平台(DMP)目标。 有关如何连接到这些目标的信息，请参阅各个链接：

* [Audience Manager DIL扩展](aam-dil-extension.md)
