# [!DNL Data Management Platform] (DMP)目标概述 {#dmp-destinations}

[!DNL Data Management Platforms] (DMP)使广告商、出版商和代理机构能够构建独一无二的受众配置文件，识别他们最有价值的区段，并在任何数字渠道中使用这些区段。 这允许您创建消费者个人资料的真实360视图，构建和激活有意义的受众。
