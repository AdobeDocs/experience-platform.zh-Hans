# 个性化目标概述 {#personalization-destinations}

## 概述 {#overview}

个性化平台从个人行为中收集数据和分析。 这些平台可帮助营销人员导航匿名数据，根据以往活动或作为具有相似习惯的受众区段的一部分向潜在客户提供优惠或广告。
