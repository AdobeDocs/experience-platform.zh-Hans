# Personalization目标概述 {#personalization-destinations}

## 概述 {#overview}

Personalization平台从个人行为中收集数据和见解。 这些平台可以协助营销人员导航匿名数据，根据先前的活动或作为具有类似习惯的受众的一部分向潜在客户提供优惠或广告。
