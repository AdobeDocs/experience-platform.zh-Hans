# Analytics目标概述 {#analytics-destinations}

## 概述 {#overview}

Analytics目标可跟踪访客与网站上内容的交互情况并收集数据以了解受众的行为。 这些平台可提供关于最有价值访客参与度和转化的独特洞察。
