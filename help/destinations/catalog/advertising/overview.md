# 广告目标 {#advertising-destinations}

将您 [!DNL Real-time Customer Data Platform] 连接到第三方广告平台(如DSP、广告网络、SSP)，并与这些平台共享假名受众。

连接到广告目标时，您的受众将作为ID发送到目标平台，在该平台中，他们将映射到目标平台所知的ID。

目前，实时CDP支持下面列出的广告目标。 有关如何连接到这些目标的信息，请参阅各个链接：
