# 广告目标概述 {#advertising-destinations}

## 概述 {#overview}

将Adobe Experience Platform连接到广告平台，例如需求端平台(DSP)、供应端平台(SSP)和广告网络，以在这些平台上共享已知和假名受众。

连接到广告目标时，受众将作为ID发送到目标平台，并在该ID被映射到目标平台已知的ID。
