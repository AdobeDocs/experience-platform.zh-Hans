# [!UICONTROL 目标]仪表板 {#destinations-dashboard}

Adobe Experience Platform用户界面(UI)提供了一个功能板，通过该功能板，您可以查看有关贵组织的活动目标的重要信息，如在每日快照期间捕获的信息。

有关如何在UI中访问目标仪表板并与之交互的详细说明，以及要详细了解仪表板中显示的可用量度，请访问[目标仪表板指南](../dashboards/guides/destinations.md)。

有关Experience Platform中所有仪表板功能的概述，请先阅读[仪表板概述](../../dashboards/home.md)。
