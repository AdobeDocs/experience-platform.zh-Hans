# Adobe Experience Platform 发行说明

**发行日期：2020 年 2 月 12 日**

Adobe Experience Platform 现有功能的更新包括：

* [源](#sources)

## 源 {#sources}

Adobe Experience Platform可以从外部源摄取数据，同时允许您使用来构建、标记和增强这些数据 [!DNL Platform] 服务。 您可以从各种来源(如Adobe应用程序、基于云的存储、第三方软件和您的CRM系统)中摄取数据。

[!DNL Experience Platform] 提供了一个RESTful API和一个交互式UI，可让您轻松为各种数据提供程序设置源连接。 这些源连接允许您进行身份验证并连接到外部存储系统和CRM服务，设置引入运行的时间，以及管理数据引入吞吐量。

**新增功能**

| 功能 | 描述 |
| ------- | ----------- |
| 对SaaS — 服务系统的API支持 | 新的源连接器 [!DNL Salesforce Service Cloud] API和 [!DNL ServiceNow] API。 |
| 针对SaaS — 营销系统的API支持 | 新的源连接器 [!DNL HubSpot] API。 |
| API支持 [!DNL NoSQL] 数据库系统 | 新的源连接器 [!DNL AWS Redshift]， [!DNL Google BigQuery]， [!DNL MariaDB]， [!DNL MySQL]， [!DNL PostgreSQL]、和 [!DNL SQL Server] API。 |
| 对云存储系统的API支持 | 新的源连接器 [!DNL Azure Data Lake Service Gen 2] API和 [!DNL Google Cloud Storage] API。 |
| 云存储系统的UI支持 | 新的源连接器 [!DNL Google Cloud Storage] 在UI中。 |
