# Adobe Experience Platform 发行说明

## 发行日期：2020 年 2 月 12 日

对Adobe Experience Platform中现有功能的更新：

* [来源](#sources)

## 来源 {#sources}

Adobe Experience Platform可以从外部源中摄取数据，同时允许您使用平台服务来构建、标记和增强这些数据。 您可以从各种来源(如Adobe应用程序、基于云的存储、第三方软件和您的CRM系统)获取数据。

Experience Platform提供RESTful API和交互式UI，使您能轻松为各种数据提供者设置源连接。 这些源连接允许您验证并连接到外部存储系统和CRM服务，为摄取运行设置时间，以及管理数据摄取吞吐量。

**新增功能**

| 功能 | 描述 |
| ------- | ----------- |
| 对SaaS的API支持——服务系统 | Salesforce Service Cloud API和ServiceNow API的新源连接器。 |
| SaaS的API支持——营销系统 | HubSpot API的新源连接器。 |
| 对NoSQL数据库系统的API支持 | 用于AWS Redshift、Google BigQuery、MariaDB、MySQL、PostgreSQL和SQL Server API的新源连接器。 |
| 云存储系统的API支持 | 用于Azure Data Lake Service Gen 2 API和Google Cloud存储API的新源连接器。 |
| 云存储系统的UI支持 | 在UI中为Google Cloud存储新增源连接器。 |
