# Personalization用例概述

Adobe Experience Platform Web SDK使企业能够为其Web资产提供各种个性化用例。 您可以跨Adobe解决方案(如Target、Journey Optimizer和Audience Manager)无缝地连接数据收集、决策和激活流程。

SDK支持灵活的体系结构，包括客户端、服务器端和混合模型，从而能够获取和呈现符合大多数组织用例的个性化优惠。

## Web SDK示例

除了此文件夹中的文档页面之外，Adobe还维护着一个您可以引用的示例应用程序存储库。 有关更多个性化用例，请参阅GitHub上的[Web SDK示例](https://github.com/adobe/alloy-samples/)，包括：

* 客户端个性化
* 服务器端个性化
* 混合个性化
* 使用单页应用程序的Personalization
