# Real-time Customer Data Platform中的目标

目标是与目标平台预建的集成，允许从Adobe Experience Platform无缝激活数据。 您可以使用目标为跨渠道营销活动、电子邮件营销活动、定向广告和许多其他用例激活已知和未知数据。

有关目标的更多详细概述（包括可用目标的列表和指向教程的链接），请阅读 [目标文档](../../destinations/home.md).
