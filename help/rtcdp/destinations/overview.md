# Real-time Customer Data Platform目标

目标是与目标平台的预建集成，可无缝激活来自Adobe Experience Platform的数据。 您可以使用目标来激活跨渠道营销活动、电子邮件促销活动、定向广告和许多其他用例的已知和未知数据。

有关目标的更详细概述（包括可用目标的列表和教程的链接），请阅读 [目标文档](../../destinations/home.md).
