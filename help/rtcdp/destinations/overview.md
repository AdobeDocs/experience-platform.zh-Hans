# Real-time Customer Data Platform中的目标

目标是预先构建的与目标平台的集成，允许从Adobe Experience Platform无缝激活数据。 您可以使用目标激活已知和未知的数据，用于跨渠道营销活动、电子邮件宣传、定向广告和许多其他用例。

有关目标的更详细概述，包括可用目标的列表以及指向教程的链接，请阅读[目标文档](../../destinations/home.md)。
