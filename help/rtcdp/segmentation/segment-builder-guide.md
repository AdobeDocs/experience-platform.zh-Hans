# Real-time Customer Data Platform中的区段生成器

基于Adobe Experience Platform构建， [!DNL Adobe Real-Time Customer Data Platform] 能够利用作为一部分的所有区段生成器功能 [!DNL Experience Platform]. 在区段生成器的用户界面中，您可以创建新区段、访问现有区段以及修改现有区段。

有关区段生成器的更详细指南，请阅读 [Segment Builder用户指南](../../segmentation/ui/segment-builder.md).
