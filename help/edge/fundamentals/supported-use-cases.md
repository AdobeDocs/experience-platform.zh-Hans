# 支持的用例

Adobe Experience Platform Web SDK可支持各种个性化用例，以及其他Adobe解决方案，例如Adobe Analytics、Audience Manager、Target和Experience Platform。

有关实施受支持用例的详细信息，请查看 [Web SDK用例项目页面](https://github.com/orgs/adobe/projects/18/views/1).
