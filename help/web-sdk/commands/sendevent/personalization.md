# `personalization`

此 `personalization` 对象包含多个属性，允许您根据用户定制用户体验。 它在Adobe Target或Adobe Journey Optimizer实施中特别有用，因为它是允许您按用户自定义显示内容的驱动力。

请参阅有关如何执行操作的文档 [呈现个性化内容](../../personalization/rendering-personalization-content.md) 以了解更多信息。
